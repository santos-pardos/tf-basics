## Terraform - S3 Example


Bucket AWS S3   

backend.tf, provider.tf, variables.tf y main.tf.

1. backend.tf (Configura el almacenamiento del estado)

2. provider.tf (Declara el proveedor)

3. variables.tf (Declara las variables de entrada)

4. main.tf (Define el recurso principal)

5. terraform.tfvars (Proporciona valores para las variables)


```
Terraform init
Terraform plan
Terraform apply
Terraform destroy
```
